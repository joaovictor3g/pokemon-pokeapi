# PokeDex 

### 💡 Project

This is an app just made for practice, using the [PokeApi v2](https://github.com/PokeAPI/pokeapi/tree/master/pokemon_v2)

### 📱 Techs
- ReactJS
- Axios

### 🔨 How to run
- Clone this repository && cd;
- `yarn` to install dependencies.
- `yarn start` to run app.
